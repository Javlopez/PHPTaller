Funciones en PHP

Es un conjunto de rutinas y acciones que a lo largo del script van a ser ejecutadas multitud de veces agrupados en una FUNCION y desde cualquier punto del script puede ser llamada y ejecutada. A su vez, esta función puede recibir parámetros externos de los cuales dependa el resultado de una función. 
<?php

function names($cadena = '')
{
  $resultado = strtolower( $cadena );
  $resultado = ucfirst( $resultado );
  return $resultado;
}

echo names('Palabras rarAS para VER como FUNCIona ');

function createList($array = array())
{
  $lis = '';
  foreach($array as $item)
  {
    $lis .= '<li>'.$item.'</li>'.PHP_EOL;
  }
  return '<ul>'.PHP_EOL.$lis.PHP_EOL.'</ul>';
}
$lista = array('lunes','martes','miercoles','jueves','viernes');
echo "<h1>Lista de dias habiles</h1>";
echo createList($lista); 
?>

Programación Orientada a Objetos(OOP)

  • Principios
    • Abstracción: La abstracción consiste en aislar un elemento de su contexto o del resto de los elementos que lo acompañan.
    • Herencia: La herencia, permite crear clases derivadas de otra clase permitiendo con ello la reutilización y la extensibilidad del software.
    • Polimorfismo: El polimorfismo permite a un mismo nombre de método representar código diferente, en consecuencia puede expresar muchos comportamientos distintos.
    • Encapsulamiento: Significa reunir a todos los elementos que pueden considerarse pertenecientes a una misma entidad, al mismo nivel de abstracción, permitiendo aumentar la cohesión de los componentes del sistema.
    • Ocultamiento:Cada objeto está aislado del exterior, es un módulo natural, y cada tipo de objeto expone una interfaz a otros objetos que especifica cómo pueden interactuar con los objetos de la clase

Programación Orientada a Objetos(OOP)

Ejemplo de una clase

<?php
class User {
  //variables/propiedades
  public $nombre;
  protected $apellido;
  private $edad;

  //constante
  const PAIS = 'Mexico';
  
  public function setData($nombre,$apellido,$edad)
  {
    $this->nombre = $nombre;
    $this->apellido = $apellido;
    $this->edad = $edad;
  }

  public function getUser()
  {
    return 'El usuario'.$this->user.' '.$this->apellido.', tiene '.
    		$this->edad.' años y es de '.User::PAIS;
  }

}

$user = new User();
$user->setData('manuel','garcia','25');
echo $user->getUser(); ?>

Arquitectura MVC

  • Modelo: Logica de negocio, es donde se gestiona la base de datos
  • Vista:Este presenta el modelo en un formato adecuado para interactuar, usualmente la interfaz de usuario.
  • Controlador:El controlador es la clase que, recibe las peticiones del usuario las procesa consultando al modelo del cual extrae datos y los envia a la vista

Metodologias y Herramientas

  • TDD(Desarrollo Guiado por Pruebas): Se basa en programar en base a requerimientos, y consiste en escribir primero la prueba y despues el codigo o refectorización.
  • Integración Continua: Consiste en hacer integraciones automáticas(Compilación y ejecución de tests de todo un proyecto) de un proyecto lo más a menudo posible para así poder detectar fallos cuanto antes.
  • Metodologias Ágiles:es un marco de trabajo conceptual de la ingeniería de software que promueve iteraciones en el desarrollo a lo largo de todo el ciclo de vida del proyecto.
